# Lead Tracker JS

A simple web based lead management system built with HTML, CSS, and vanilla JavaScript.

## Live Demo
https://sagar08072003.github.io/lead-tracker-js/

## Features

* Add new leads
* Edit existing leads
* Delete leads
* Data saved using browser localStorage
* Works after page refresh
* Simple, clean UI

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
* Browser localStorage
